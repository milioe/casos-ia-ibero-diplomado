# Avatares de Herramientas

Esta carpeta contiene los iconos que se mostrarán cuando las herramientas se ejecuten en el chat.

## Iconos necesarios (archivos PNG):

1. **obtener_info_cliente.png** - Para consultar información del cliente
2. **obtener_contratos_cliente.png** - Para consultar contratos
3. **agendar_cita.png** - Para agendar citas
4. **enviar_factura_contratos.png** - Para enviar facturas

## Especificaciones:
- Formato: PNG
- Tamaño recomendado: 40x40 px o 64x64 px
- Fondo: Transparente preferiblemente
- Los nombres deben coincidir exactamente con el nombre de la herramienta

## Opciones para obtener iconos:

### Opción 1: Usar generadores online
- https://www.flaticon.com - Buscar iconos y descargar PNG
- https://icons8.com - Biblioteca de iconos gratis
- https://www.iconfinder.com - Iconos variados

### Opción 2: Convertir emojis a PNG
- https://emoji.gg - Descargar emojis como PNG
- Sugerencias:
  - obtener_info_cliente: 👤 o 📋
  - obtener_contratos_cliente: 📄 o 📑
  - agendar_cita: 📅 o 🗓️
  - enviar_factura_contratos: 💳 o 📧

### Opción 3: Usar Font Awesome o Material Icons
- Descargar el icono como SVG
- Convertir a PNG usando: https://cloudconvert.com/svg-to-png
